Kokoro TTS Launcher

Launch Kokoro TTS API

Click the button below to launch the Kokoro TTS API in Google Colab. This will open a new browser tab.

Launch Kokoro TTS in Colab

Important Steps:

  1. After Colab opens, click on "Connect" to connect to a runtime (preferably with GPU)
  2. Then click "Run cell" to execute the code (or press Shift+Enter)
  3. Wait until you see "Kokoro API is now available at: [URL]" message
  4. Copy that URL and use it as your TTS endpoint

Using the API

Once the API is running, you can use it with this endpoint:

https://[your-cloudflare-url].trycloudflare.com/v1/audio/speech

Example curl command:

curl -X POST https://[your-cloudflare-url].trycloudflare.com/v1/audio/speech \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kokoro",
    "input": "Hello, this is a test of Kokoro TTS!",
    "voice": "af_bella",
    "response_format": "mp3"
  }' \
  --output test_audio.mp3

Keep Colab Running

To prevent your Colab session from disconnecting automatically, you can run this JavaScript in your browser console:

function ClickConnect() {
  console.log("Clicked connect button");
  document.querySelector("colab-toolbar-button#connect").click();
}
setInterval(ClickConnect, 60000);
